شاشة ملف موظف كاملة واحترافية جداً.

التصميم:
┌──────────────────────────────────────────────────────┐
│ Header: [← العودة] ملف الموظف #ID  [حذف][طباعة][حفظ]│
├──────────┬───────────────────────────────────────────┤
│ Sidebar  │ Content                                    │
│ 320px    │                                            │
│          │ Tabs: [شخصية|وظيفية|راتب|مستندات|سجل]     │
│ صورة     │                                            │
│ الاسم    │ Cards with form fields (3-column grid)     │
│ المسمى   │                                            │
│ الرقم    │ Each section = collapsible card            │
│          │ Fields: input, select, date, readonly      │
│ 4 Stats  │                                            │
│ التواصل  │                                            │
└──────────┴───────────────────────────────────────────┘

التابات:
1. البيانات الشخصية:
   - الاسم (أول/أب/عائلة) + الاسم بالإنجليزي
   - الجنسية (dropdown من جدول nationalities)
   - تاريخ الميلاد + الديانة + الحالة الاجتماعية + عدد الأبناء
   - رقم الإقامة/الهوية + تاريخ الإصدار + الانتهاء
   - رقم الجواز + تاريخ انتهاء الجواز + رقم الحدود

2. البيانات الوظيفية:
   - الرقم الوظيفي (readonly)
   - المسمى الوظيفي (dropdown من job_titles)
   - القسم (dropdown من departments)
   - المدير المباشر
   - تاريخ التعيين + نوع العقد + موقع العمل + نظام الدوام

3. الراتب والمستحقات:
   - جدول: البند | النوع | المبلغ
   - الأساسي + بدل سكن + بدل نقل + بدلات أخرى
   - خصومات (تأمينات، سلف)
   - صافي الراتب (مجموع تلقائي)
   - تقدير مكافأة نهاية الخدمة

4. المستندات:
   - كروت بأيقونات حسب نوع الملف (PDF/IMG/DOC)
   - اسم الملف + الحجم + التاريخ
   - زرار إضافة مستند جديد

5. السجل الوظيفي:
   - Timeline عمودي (ترقيات، نقل، تدريب، تعيين)
   - التاريخ + العنوان + التفاصيل

Data Integration:
- عند فتح الشاشة: bridge.get_employee(id) يجيب كل البيانات
- الـ Dropdowns: bridge.get_dropdown_data("departments") etc.
- زرار الحفظ: يجمع كل البيانات ويبعتها bridge.save_employee(json)
- Toast notification عند الحفظ الناجح

التصميم:
- Dark theme (نفس الألوان في CLAUDE.md)
- Cairo + JetBrains Mono fonts
- RTL كامل
- أنيميشن ناعم (slide-up, fade)
- Hover effects على الكروت والأزرار
- الشاشة لازم تكون BEAUTIFUL - دي showcase للنظام الجديد
